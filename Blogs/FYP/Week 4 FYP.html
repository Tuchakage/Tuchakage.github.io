Week 4 - Finishing The Bow & Arrow Pt 1

Date Created: Nov 24, 2021

Setting Up String For Bow!

After doing the tasks to pick up and drop the bow and arrow it was time to get started on actually launching the arrow. To get the Bow to launch the arrow we first needed some way for the player to be able to draw back and release the string, for this we used a box collision that the string of the bow will follow.

Since the string that was already on the Bow model wont move at all since the bow is a static mesh in the project instead of a skeletal mesh (which it was with the VR expansion plugin) i had to edit the actual model in Maya to get rid of the string.

In Maya i had to also to change the inital rotation of the bow to be stand up right so that i could properly put on the cable component which is what we are using as the string instead. The reason for using the cable component in Unreal as the new string for the bow is that the cable component can follow the String collision (The Box collision component) so it actually looks like the player is drawing back the string of the bow (Look at "Picking Up The Bow" Image down below).

How Pulling Back The String Works

The way pulling back the string is setup is that i had to first make it so that the first hand that touches the bow and picks it up will be the first hand and then once the bow has been picked up you can then use your second hand to pull the string back (in this case the box collision).

When the string is grabbed with the second hand then the Event tick is enabled which will then track the location of the second hand which the string will then follow, it will also make sure that the string cannot go too far forward and too far back (Look At "Tracking The Second Hand" Image Below).

For the String to go back into place we created a custom event called "Shoot" that used a timeline to make the string smoothly go back into its original position and this event is called when the second hand stops holding the string.

The power at which the arrow has when its launched is determined when you get the vector length of the distance between the original location of the string and the current location of the string.
