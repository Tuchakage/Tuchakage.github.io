Week 13 Final Meetup With Mentor

Date Created: Apr 9, 2022

Meeting With Mentor

I had one last meeting with my mentor just to check up on my game and see what i should add to finish off the game. He said that i should add a map and add a death animation to finish the project which are things i could do quickly.

For the map i just found some assets that i found in my Epic Games library and then i just built the map from there and as for the death animation for the Enemy that was simple to add, just required putting the animation in the state machine and a variable that tells the animation blueprint when to change into that animation. Since a map was added the spawnpoints of the Enemy was changed to be inside of the gates.

Text in the arena was also added so that the player can see what wave number is on, the way it works is that it will get the wave number from the VR_Gamemode Blueprint created before and then it will keep updating itself. When there is no more waves left (you beat all the waves and won) then the text changes to "Winner".

I created a weapon spawner so that the Sword can spawn in when its Wave 2, i did this because the Sword is really strong when compared to the Bow and the Dagger. The way the Weapon Spawner works is exactly like how the Enemy Spawner works with the SpawnWeapon function being binded to the NextWave function inside the VR_Gamemode Blueprint.
