Week 9 - UI Update & Publishing Time!

Date Created: Dec 6, 2021

New Mini Game Idea!

Me, Hassan and Shaafie were thinking to ourselves how it was strange that are team name is PackingUp, our game is named Donut pack but yet, in our game theres nothing that includes packing things up. So we were trying to think of an idea for a packing mini game which is when i remembered how there was a mini game in one of the old Mario DS games where you have to launch a cannon ball from a slingshot into the bombs that were floating down in the air.

In our version of the mini game i told them that we could be launching Donuts from a sling shot into a package for the Donut instead, Hassan and Shaafie started playing the mini game since they have the game and they agreed that we should make a new mini game based off this.

Starting On The Slingshot!

I started to add in a way to drag and shoot but the problem with it was that it doesn't feel like it was coming out of a sling shot, it felt more like your shooting out a rocket.

I then stumbled upon a Youtube tutorial that was showing viewers how to make an Angry birds type sling shot, the only problem was that this was for PC. So when following the tutorial i had to adapt it from PC to touch screen and to be honest it wasn't really that hard because i just had to change the input from left clicking to the touch screen input of mobile.

Respawning Ball

I made some changes to how the ball is respawned, in the video it is respawned by adding multiple ball objects and just disabling them in the editor Scene but i want the balls to be spawned by a script so i can edit how many balls will be spawned later on.

I made it so that the nextDonut variable that is already there will be set to the GameObject it is on when the game starts and then i created a new script called "SpawnDonut" which essentially just spawns the next Donut with all the components reset. So i created a new function that calls this function in the new script "SpawnDonut" and then once that function is called it will then make the band for the slingshot visible again and then the Donut that has already been launched will be destroyed.
