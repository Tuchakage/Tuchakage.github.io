Week 8 - Finale...

Date Created: May 2, 2022

Problem Caused By Branches

The deadline for the project was very close so all of us had to go into overdrive but there was a problem with the Ranged Enemy BP where it couldnt find its parent class which is the BasicEnemyBP. Me and the other gameplay programmers believe this had happened because of the branches because after merging into the main branch this problem would appear, to fix this one of the gameplay programmers just recreated the Ranged Enemy.

Finishing All The Cutscenes

Luckily i was able to keep up with the pace of the storyboard creation somewhat, whenever a new storyboard was made i would work on it straight away and eventually finished all of the storyboards resulting in all the cinematics being done. In between waiting for the storyboards i created the texture to create lightning and then imported that into Unreal to create the lightning particles for the main character which would be used for both cutscenes and gameplay.

Final Voice Acting Session

The last thing to do to complete the cinematics side of things was to record all the lines, since the script was done me and a few others who were voice acting met up in Uni to record all the lines, we even managed to get a female voice from someone whos in our class but not in our team, once again the session was a success.

All that was left was for the voice lines to be edited and organised, where i would then help out the audio engineer add the voice lines into the remaining scenes that didnt have them. We were going to have sound effects in the game but there was problem with the gameplay so i had to go and help the gameplay programmers out with that as well. Since the game is big it would have taken ages to build the game so we just had to skip the sound effects for now.
