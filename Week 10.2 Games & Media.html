Week 10 - Fixing A Few Problems I Found

Date Created: Dec 16, 2021

Problem With Packing Mini game

I noticed how in the Packing Mini game when you shoot another Donut at a package that is already closed, you will still get points from it. To fix this all i had to do was go to the script that handles the collision between the Package and the Donut and i just had to disable the box collider on the Package after its sprite has changed to being closed (Being hit by the Donut).

Problem With Containers Back For Round 2.....

There was another problem i found whilst play testing the game and that was that the previous problem where when a container was already full but then the player puts the wrong ingredient inside of it but the game will still think its full came back, im pretty sure this problem came back because the amount of items needed to fill up the container has been changed.

My solution to this problem is that i created a function that everytime the wrong ingredients is put into the wrong container the function will check if the "isComplete" variable (Which i put there at the beginning of the project but was never used) is true and if it is then it will tell the game manager that one container is no longer complete and it will set "isComplete" but to false.
