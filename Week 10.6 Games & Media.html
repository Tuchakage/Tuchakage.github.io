Date Created: Dec 12, 2021

Final Rating

So in our game so far we have the Rating System but there was no way the Player could know their overall performance in the level, that is until i created a scene that will show the overall rating the played got for the level.

I created a function called "FinalRatingCalc" that when the Final Scene is loaded it will get all the ratings of the previous mini games and it will add it all together. Depending on the total score the SetFinalRating() function will be called with an int parameter passed into it (A number between 1 and 3 will be passed into these functions).

What the SetFinalRating() function does is that it will display the amount of stars in the Final Rating Scene and it will also create save data for the level so that the player can see what rating they got in that level when they are in the main menu.

Understanding How A Saving System Works

The Saving system is something that i've always wanted to add into a game but i never really knew how to even make one, until i decided to try it out for this game.

Implementing the Saving system was pretty hard at first because i saw that you have to understand how to read and write data into files, after some research i was able to understand the process. The hardest part was how to save a dictionary into a file, because since dictionaries arent serializable i cannot just easily save them into a file like an integer or a string, but luckily i was able to find a script that converts dictionarys into string.

I had to modify this script so that it could take in a parameter so that the save system knows what level they are making a save file for and what the final rating of that level is, i also had to add in a separate load function so that i choose when the save data is loaded.
